### About

Gittydocs turns a folder of Markdown/MDX files on GitHub into a documentation website. I wanted something where you just organize your content and deploy — no config headaches.

Built with Velite, Vike, and SolidJS.

- **Full-text search** (Ctrl+K)
- **Auto-generated table of contents** with scroll tracking
- **Dark mode** out of the box
- **One-click page copying** as Markdown
- **"Edit this page" links** that go straight to GitHub
- **Auto-generates llms.txt** for LLM-friendly output
- **Ejectable** — customize the stack if you need to
