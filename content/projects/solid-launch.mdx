### About

Solid Launch is my opinionated full-stack starter for SolidJS + Vike + Hono. It's basically everything I wish came out of the box when starting a new project.

- **SolidJS** for the frontend
- **Vike** for file-based routing
- **Hono** as the backend API layer
- **SQLite** for serverless-ready persistence
- **Session-based auth** ready to go
- **Billing/subscription scaffolding** for SaaS

I use this as my go-to starting point for new projects. It skips all the boring setup so I can focus on the actual product.
