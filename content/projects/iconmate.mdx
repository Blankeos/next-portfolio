### About

Iconmate is a CLI tool for adding SVG icons to JS projects without pulling in entire icon libraries. I got tired of copy-pasting SVGs or importing massive packages just for a handful of icons, so I built this.

You can fetch icons from Iconify, URLs, or raw SVG, and it generates framework-specific components (React, Vue, Svelte, Solid, or vanilla SVG).

- **Interactive terminal UI** for browsing and selecting icons
- **Non-interactive mode with JSON output** for scripting and coding agents
- **Per-project and global config** with JSON schemas
- **Written in Rust** — fast, standalone binary, no Node.js needed
