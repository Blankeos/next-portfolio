### About

SpacetimeDB Studio is a local web UI for managing SpacetimeDB databases — like Prisma Studio or Drizzle Kit Studio but for SpacetimeDB. Built with SolidJS and Vike.

- **SQL editor** with syntax highlighting and autocompletion
- **Vim keybindings** (toggleable) for power users
- **Inline editing** — edit query results directly, auto-generates UPDATE statements
- **Schema browser** — tables, columns, indexes, reducers
- **One command to launch** — `npx spacetime-studio <database>` opens at localhost:5555

Built as a Bun monorepo with a SolidJS+Vike web app and a CLI distributed via npm.
