### About

Modelcli gives you access to any LLM straight from the terminal. I built it in Rust so it's fast and works as a standalone binary. Supports OpenAI, Anthropic, Google, xAI, Groq, Mistral, and any OpenAI-compatible provider.

- **Streaming responses** in real-time
- **Reasoning token support** for thinking/reasoning models
- **JSON output mode** for scripting
- **Piping support** — chain it with other CLI tools
- **Custom providers** — add any OpenAI-compatible endpoint

I use this mainly to power AI commit messages in lazygitrs and for quick queries without leaving the terminal.
