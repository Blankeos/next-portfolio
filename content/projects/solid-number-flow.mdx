### About

Solid Number Flow is a SolidJS port of number-flow. It smoothly animates numbers when values change — useful for dashboards, counters, price displays, that kind of thing.

- **Smooth transitions** when values update
- **Number formatting** — decimals, thousands separators, etc.
- **Localization** — respects locale settings
- **Zero config** — just pass a number and it animates

Built for SolidJS's fine-grained reactivity. It's a small thing but it makes UIs feel a lot more polished.
